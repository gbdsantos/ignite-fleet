## ⚙️ Getting Start

```Bash
# Install dependencies
npm install

# Build project
npx expo prebuild

# Run
npx expo start

# Android
npx expo run:android
```

Attention with API Keys values added in below files with expo prebuild command:

- AndroidManifest.xml (android folder)
- AppDelegate.mm (ios folder)
- Info.plist (ios folder)
